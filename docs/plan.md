# EYE-D 프로젝트 개발 계획

본 문서는 **EYE-D (실내 인원 추적 관리 시스템)** 프로젝트가 로컬 더미 환경을 넘어 실하드웨어, 분산 네트워크, 프로덕션 성능 튜닝 단계로 스케일업하기 위한 향후 개발 작업 목록을 정리한 문서입니다.
